## LLM Context Priming Experiment

This repository contains the code and outputs for a small experiment exploring how preceding context can influence the responses generated by large language models (LLMs).

# Experiment

The model was given the following prompt:

Create an investment strategy with the goal of beating the S&P 500 over the next year.

The prompt was then preceded by one of several passages:

Control: No additional context
John Galt (Atlas Shrugged)
Warren Buffett
Jim Simons

The objective was not to evaluate the quality of the investment strategies. Instead, the goal was to observe whether adding different contextual passages changed the model's subsequent reasoning.

# Methodology

Model temperature was set to 0 to minimize sampling variance.
Each condition was run multiple times to verify reproducibility.
The same investment prompt was used across all conditions.
Only the preceding passage was changed.
Observations

At a high level, the model consistently produced similar investment frameworks involving research, diversification, risk management, and portfolio monitoring.

However, the surrounding context appeared to influence the model's framing:

Control: Generic investment advice.
John Galt: Greater emphasis on innovation, growth, active management, and technology.
Warren Buffett: Increased focus on discipline, process, and long-term thinking.
Jim Simons: A shift toward quantitative analysis, machine learning, pattern recognition, and systematic investing.

# Takeaway

This experiment suggests that LLM outputs can be simultaneously:

Robust in their overall structure, and 
Sensitive to contextual priming that changes how problems are framed and approached.

As AI systems are increasingly used for research, analysis, and decision support, understanding how context shapes model behavior may become as important as improving the underlying models themselves.
